# Anderson Quintanilha — Site Pessoal

Landing page pessoal de Anderson Quintanilha, Arquiteto de Sistemas Inteligentes.

## Páginas

- **`index.html`** — Landing page principal com links e apresentação
- **`orcamento.html`** — Questionário de orçamento multi-etapas (8 seções)

## Tecnologias

- HTML5 semântico
- CSS puro (dark mode · neon gold · glassmorphism)
- JavaScript vanilla (partículas, animações, webhook)
- Google Fonts: Inter + Space Grotesk

## Funcionalidades

- ✨ Partículas animadas com atração ao mouse
- 🎨 Design dark mode com paleta gold
- 📋 Questionário de orçamento com 8 etapas
- 🔗 Envio automático para webhook n8n
- 📱 Totalmente responsivo
- ♿ Acessível (aria-labels, focus visível)
